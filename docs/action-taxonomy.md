# Action Taxonomy Reference

## Speech Acts (Searle)
What the AI is doing communicatively.

| Act | Definition |
|-----|-----------|
| **assertive** | Stating facts, sharing information |
| **directive** | Requesting action, asking questions, guiding |
| **expressive** | Expressing emotion, validating, affirming |
| **commissive** | Making a promise, committing to action |
| **declarative** | Changing state (e.g., "We're done here") |

## Dialogue Acts (Domain-Specific)
What the AI is doing in this hiring conversation.

| Act | Definition |
|-----|-----------|
| **open_with_question** | Starting conversation, initial probe |
| **probe_deeper** | Following up on what person said, exploring further |
| **ask_for_concrete** | Requesting specific examples vs. abstractions |
| **validate_genuine** | Confirming authenticity, mirroring understanding |
| **redirect_from_surface** | Moving past vague language or performance |
| **reflect_understanding** | Summarizing or restating to clarify |
| **affirm_commitment** | Supporting a decision or insight |

## Rubric Scores
How well does the AI's **response** help evaluate the person on each criterion? (1-10 scale, null if not evaluated)

- **depth-of-questioning**: Does this turn help explore thinking depth?
- **self-awareness**: Does this help gauge articulate self-understanding?
- **systems-thinking**: Does this probe community/personal connections?
- **experimentation-evidence**: Does this reveal if they build vs. passive?
- **authenticity**: Does this expose genuine vs. performing?
- **reciprocal-curiosity**: Does this invite mutual curiosity?

## Fit Score
0-100 overall quality of this turn (null if not evaluated).
